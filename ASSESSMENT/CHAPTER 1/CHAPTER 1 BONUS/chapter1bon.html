HTML & CSS Quick Reference Guide

HTML Tags

<p> - Paragraph

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a> - Anchor (Link)

Creates a hyperlink to another page.

<a href="https://example.com">Visit Example</a>

<img> - Image

Displays an image.

<img src="image.jpg" alt="Description">

<ul> & <li> - Unordered List & List Item

Creates a bulleted list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table> - Table

Defines a table with rows and columns.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

CSS Properties

color

Sets the text color.

p {
    color: blue;
}

background-color

Defines the background color of an element.

body {
    background-color: #f4f4f4;
}

font-size

Specifies the font size of text.

h1 {
    font-size: 24px;
}

margin

Sets the outer spacing around an element.

div {
    margin: 20px;
}

padding

Defines the inner spacing inside an element.

div {
    padding: 15px;
}

border

Sets the border style, width, and color.

div {
    border: 2px solid black;
}
